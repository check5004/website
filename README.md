# Personal Static Websites

私が個人的に簡単に作った静的Webサイトを公開・管理するためのリポジトリです。

## 概要

このリポジトリでは、ちょっとしたツールやWebページを静的サイト（HTML / CSS / JavaScript）として作成し、管理しています。
機能やプロジェクトごとにディレクトリを分け、それぞれのディレクトリ内に `index.html` を配置する構成をとっています。

## デプロイ・公開について

このリポジトリのWebサイトは **GitHub Pages** を利用してホスティングしています。
GitHub Actions (`.github/workflows/static.yml`) を設定しており、`main` ブランチにプッシュされると自動的に `./public` ディレクトリ配下のコンテンツが公開される仕組みです。

* **公開URL (例):** `https://check5004.github.io/website/`

## サイト一覧（ディレクトリ構成）

現在の `public/` 配下には以下のサイトが格納されています。

* **`net-pay-calculator/`**
  * **サイト名:** 給与・残業手取りシミュレーター
  * **概要:** 労働条件通知書や給与明細の数値を基に、残業時間、定期代、ボーナスなどを加味した予測手取り額をブラウザ上で計算できるツールです。
  * **アクセス先:** `https://check5004.github.io/website/net-pay-calculator/`

## 今後の予定

今後も新しく個人的な静的サイトやツールを作った際は、`public/` 配下に新しいディレクトリを切って随時追加していく予定です。
